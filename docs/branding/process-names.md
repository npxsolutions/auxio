# Process-grounded names — what actually happens to the data

Your product is a **3-step transformation**:

```
IN          TRANSFORM          OUT
Shopify  →  Validate           →  eBay
channels →  Translate          →  Amazon
APIs     →  Enrich             →  TikTok Shop
feeds    →  Standardise        →  Etsy
         →  Route              →  Walmart ...
```

**Raw catalogue data goes in. Marketplace-ready listings come out.** Names that capture that middle step — the transformation itself — are the most ownable.

Best metaphors: **refinery · mill · forge · kiln · press · atelier**. Each describes a place where raw input becomes something sellable.

---

## Whole-process metaphors (the strongest — name the loop itself)

Short names where the word itself describes "raw goes in, finished comes out":

| Name | Evokes |
|---|---|
| **Forge** | Heat + shape. Strong. Feels like software infrastructure. |
| **Mill** | Grain in, flour out. Every SKU ground to marketplace grade. |
| **Kiln** | Clay in, pottery out. Finished, fired, ready. Unclaimed in SaaS. |
| **Press** | Raw paper + ink → published pages. Also "deadline pressure". |
| **Foundry** | Molten metal cast into shape. Technical. |
| **Refinery** | Crude in, refined product out. Chemical/oil resonance. |
| **Crucible** | Where transformation happens under heat. Distinctive. |
| **Smithy** | Craft workshop. Warm + ancient. |
| **Atelier** | Fine craft studio. Elegant but European. |
| **Anvil** | Where shape is struck. One syllable. Strong. |
| **Distillery** | Essence extracted. Longer but poetic. |
| **Mint** | Where currency is made. "Minted" = polished + official. |

My instinct for the top 3: **Forge · Kiln · Mint**. Two-letter variations also:
- **Forgeworks**
- **Kilnhouse**
- **Mintline**

## Step 1: ingest / inflow (data coming in)

Names that lean into the gathering side:

Intake · Uptake · Inflow · Harvest · Source · Catch · Gather · Absorb · Siphon · Draft · Draw · Quarry · Wellhead · Stream · Funnel · Manifold · Intake · Gateway · Inlet · Conduit · Pipe · Tributary · Feeder

**Picks**: *Manifold* (plumbing + mathematical richness), *Wellhead* (source), *Tributary* (rivers feeding the main channel — evokes multi-source commerce)

---

## Step 2: transform / the middle — where your product lives

Names that lean into the shaping/refining act:

Refine · Render · Transmute · Knead · Shape · Mold · Temper · Smelt · Cast · Forge · Hone · Polish · Burnish · Sift · Strain · Filter · Distill · Anneal · Craft · Carve · Chisel · Plane · Prune · Trim · Whittle · Press · Stamp · Seal · Mark · Imprint · Engrave · Etch · Score · Carve · Weave · Stitch · Ply

**Picks**: *Render* (3D graphics + old "render unto" — produce a finished output), *Temper* (strengthen through controlled heat — perfect for a validator), *Anneal* (soften + reshape — quietly technical), *Imprint* (stamp the marketplace's shape onto the data)

---

## Step 3: dispatch / outflow (send to marketplaces)

Names that lean into the distribution side:

Dispatch · Relay · Deploy · Publish · Issue · Emit · Cast · Launch · Route · Deliver · Carry · Ferry · Haul · Convey · Transport · Ship · Send · Post · Broadcast · Signal · Herald · Courier

**Picks**: *Ferry* (carries + old commerce), *Courier* (delivery + fast + recognisable), *Dispatch* (authoritative, logistics-native)

## The honest top 8 — names I'd actually live-verify

After filtering for (a) captures the whole transformation, (b) short enough to be memorable, (c) feels like software in 2026, (d) low-known-collision heuristic:

1. **Forge** — near-certainly taken in some form but has multiple variants (Forgehq, Forgeline, Forgepress); the metaphor is so clean it's worth checking
2. **Kiln** — unclaimed in the SaaS space I know of, short, evocative
3. **Crucible** — where transformation happens; distinctive
4. **Temper** — rare, dual meaning (strengthen steel + restraint), 2 syllables
5. **Mintline** — coined compound, mint = produce officially + line = pipeline
6. **Render** — graphics + commerce "render services"; may be taken
7. **Anneal** — soften + reshape; distinctive; might be too obscure
8. **Ferry** — delivery between places; short; maritime-commerce resonance
