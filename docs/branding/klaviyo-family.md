## The test

For each name that catches your ear:

1. **Say it 3 times out loud** — does it still feel good by the third?
2. **Say "Welcome to [name]"** — does it sound like a real company?
3. **Spell it on a phone call** — would the other person get it right first time? ("M-O-R-R-I-O, like Mario with two Rs")
4. **Google-ability** — is there an obvious 1-result dominance if you owned it?

If a name passes all 4 — circle it.
