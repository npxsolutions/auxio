# Hooks library

100 founder-voice hooks for Meridia, grouped by format. Each hook is the **first two lines** that show before "see more" on LinkedIn or the first tweet in a thread. Pick one, write the 250 words underneath, post.

All written in the same voice as `linkedin-posts.md`: no exclamation marks, no emoji, no "game-changing" / "10x" / "supercharge". Editorial, founder-direct, specific numbers, real operator friction.

## How to use this library

1. Pick a hook that matches your week's rhythm (Narrative / Tactical / Product / POV)
2. Write 250 words underneath — specific numbers, one idea, one operator friction
3. Add 3–5 hashtags (use the ones from `linkedin-posts.md` by category)
4. Schedule for a Tuesday / Thursday / Friday at 08:30 / 09:15 / 10:00 UK (operator-hours)
5. Reply to every comment for the first six hours — that's where the algorithm pushes

## To generate fresh hooks

Open a Claude / ChatGPT chat, paste this library + `docs/marketing/positioning.md`, and ask:

> Draft 10 new LinkedIn hooks for Meridia matching the voice of this library. Each hook is two lines — an assertion and a follow-up that creates tension. Focus on [narrative | tactical | product | POV]. Use real numbers where I've seen them. No exclamation marks. No emoji. No "game-changing" or "10x".

Expect to reject 6 of 10. Keep the 4 that land.
