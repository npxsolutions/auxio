# Ops

Scaling-the-company docs, not scaling-the-product docs. Kept deliberately high-level — these aren't runbooks.

## Files

- `hiring.md` — first 10 hires by seniority + function, comp bands, equity philosophy, interview loops, remote/hybrid stance, where we source.
- `exit-strategy.md` — optionality we're preserving. Strategic acquirer list (Shopify, Intuit, Adobe, Oracle NetSuite, SAP, Klaviyo, Fairing), IPO viability thresholds, founder incentives, cap-table hygiene notes.

## Key questions

- Hiring: what is the first hire that buys back founder time most cheaply?
- Exit: what multiple of ARR do comps in this space clear, and what metrics move that multiple?
- What decisions today foreclose options later (e.g. equity structure, debt, exclusivity deals)?
